# Selenium Practice HTML Forms

This repository contains simple HTML forms created specifically for **Selenium automation testing**.

## Files
- `form.html` – Basic HTML form with input fields, radio buttons, dropdown, and submit button.
- `selenium.html` – Selenium-friendly form with proper `id` and `name` attributes.

## Purpose
- Practice Selenium IDE recording
- Practice Selenium WebDriver automation
- Learn element locating strategies (id, name, xpath, cssSelector)

## How to Use with Selenium IDE
1. Host the files using a local or GitHub Pages server.
2. Open the page using `http://` or `https://`.
3. Start recording in Selenium IDE.

## Technologies Used
- HTML
- CSS
- Selenium IDE / Selenium WebDriver
